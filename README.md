# 🟢 Green Streak

This repo automatically commits every day using GitHub Actions.

No server needed. No manual work. Just green squares on my GitHub profile — every single day.

## How it works
A GitHub Actions workflow runs daily at 9AM UTC, appends today's date to `log.txt`, and pushes the commit automatically.
